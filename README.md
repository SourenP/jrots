## Run

1. Start [atom](https://atom.io/)
 - install [atom-hydra](https://atom.io/packages/atom-hydra)
2. Toggle hydra
 - packages > atom-hydra > toggle
3. Open, select all and shift-enter these files in order:
 - `midi_1.js`
 - `draw_2.js`
 - `letter_3_1.js`
 - `hydra_4.js`
